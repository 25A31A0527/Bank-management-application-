# Bank Management Application (C++)

## Introduction
This project is a console-based Bank Management Application developed in C++.  
It simulates basic banking operations using object-oriented programming and file handling.

---

## Features
- Create new bank accounts
- Deposit and withdraw money
- Check account balance
- Search account records
- Store customer data using file handling
- Menu-driven application

---

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

---

## How the System Works
1. User selects an option from the menu.
2. Account details are entered.
3. Data is stored permanently in a text file (`bank_data.txt`).
4. User can search and display account records anytime.

---

## File Used
- `bank_data.txt` → Stores all bank account details permanently.

---

## Menu Options
1. Create New Account
2. Display All Accounts
3. Search Account
4. Exit

---

## Advantages
- Easy to use
- Permanent record storage
- Secure data handling
- Beginner-friendly C++ project
